jive motion script ω version

this script read playlist text file and make it into youtube playlist.
when you have hundreds of song and singer name listed text and want them into the playlist, this script will be your help.

you need to have your web server, google account and some experience about www and FTP.


1. download source codes from github https://github.com/konbuerimaki/text2playlist.
2. log in google with your acccount.
3. open google developer console. https://console.developers.google.com/
4. create OAuth2.0 client ID in API Manager. choose "web application".
5. IMPORTANT! specify your web server URL in "authorized JavaScript origins".
6. no need to specify "authorized redirect URIs".
7. copy client ID and paste it in modernjiveplaylisting.js.
8. location to copy is a line 28 "clientId".
9. paste your playlist text file URL at a line 49 "_URL".
10. create a blank youtube playlist.
11. paste that playlist id(you can find it in a playlist url) at a line 50 "_playlist_id".
12. upload updated source codes to your own web server.
13. open a index.html on the browser which logged in google account that you made this client Id. buttons are activated if OAuth2.0 is working.

now it shuold work.


for security, after you made a playlist. it would be good to delete your API client id.
read about your API Key security. https://support.google.com/googleapi/answer/6310037?hl=en&ref_topic=7013279
